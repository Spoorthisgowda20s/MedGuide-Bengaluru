# 🩺 MedGuide-Bengaluru

MedGuide-Bengaluru is a healthcare recommendation web application designed to help users find suitable doctors across Bengaluru through an intuitive and user-friendly interface. The system utilizes the **Bengaluru Doctors Dataset** collected from Practo and provides doctor recommendations based on user preferences.

##  Features

*  Search and explore doctors from Bengaluru.
*  Access information on **3,644 doctors** across **29 medical specialties**.
*  Browse doctors from **34 different locations** in Bengaluru.
*  View doctor rankings and recommendations generated using machine learning techniques.
*  Interactive and responsive user interface for a seamless user experience.
*  Simplifies the process of finding healthcare professionals based on user needs.

##  Technologies Used

* Python
* HTML, CSS, JavaScript
* Machine Learning
* Pandas & NumPy
* Pickle
* CSV Dataset Processing

##  Dataset

The project uses the **Bengaluru Doctors Dataset** sourced from Practo, containing details such as doctor specialties, locations, and other relevant information required for recommendation and ranking.

##  How to Run the Project

1. Clone this repository.
2. Download the project files.
3. Install the required dependencies.
4. Run the application on your local machine.
5. Open the application in your browser to explore doctor recommendations.

##  Objective

The primary objective of MedGuide-Bengaluru is to make healthcare discovery easier by helping users identify suitable doctors quickly and efficiently through data-driven recommendations.

##  Team Project

This project was developed as part of an academic initiative to demonstrate the application of machine learning and web technologies in solving real-world healthcare accessibility challenges.
